# WebApp书城整站开发 

## ✏️ 项目初始化
下拉代码     
`$ git clone xx`   
打开根目录初始化   
`$ npm install`   
启动服务器   
`$ node app.js`   
打开以下网址即可   
http://127.0.0.1:3000/
