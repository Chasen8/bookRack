# WebApp书城整站开发 

## ✏️ 项目初始化
下拉代码     
``$ git clone xx``   
打开根目录初始化   
``$ npm install``   
启动服务器   
``$ node app.js``   
打开以下网址即可   
http://127.0.0.1:3000/   

## ✏️ 收获

1. 了解了Web APP开发现状；
1. 服务端环境搭建和开发过程；
1. 了解了koa、Vue、EJS模板的使用；
1. 知道base64图片制作使用；了解了一些HTML5的API；
1. 了解页面基本结构开发过程；交互代码开发过程；页面和服务器通信代码的开发；
